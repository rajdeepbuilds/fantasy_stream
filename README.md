**API:** The Movie Database

**Deployment:** Vercel




## Run Locally

Clone the project

```bash
  git clone https://github.com/rajdeepbuilds/fantasy_stream.git
```

Go to the project directory

```bash
  cd fantasy_stream
  cd fantasy-stream-app 
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
